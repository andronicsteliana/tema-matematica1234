Puteri și Radicali

Clasa a 9-a - Formule și proprietăți

I. Proprietăți ale Puterilor

1. Înmulțirea puterilor cu aceeași bază

am · an = am+n
Se păstrează baza și se adună exponenții

2. Împărțirea puterilor cu aceeași bază

am / an = am-n
Se păstrează baza și se scad exponenții

3. Ridicarea unei puteri la putere

(am)n = amn
Se păstrează baza și se înmulțesc exponenții

4. Ridicarea unui produs la putere

(ab)n = an · bn
Se ridică fiecare factor la puterea respectivă

5. Ridicarea unui raport la putere

(a/b)n = an / bn
Se ridică atât numărătorul cât și numitorul la puterea respectivă

6. Puteri cu exponenți speciali

a0 = 1 (a ≠ 0)
a1 = a
a-n = 1 / an (a ≠ 0)

II. Proprietăți ale Radicalilor

1. Radicalul unui pătrat

√(a²) = |a|
Rezultatul este modulul numărului (valoarea absolută)

2. Radicalul unui produs

√(ab) = √a · √b
Radicalul unui produs este produsul radicalilor (a, b ≥ 0)

3. Radicalul unui raport

√a / √b = √(a/b)
Împărțirea radicalilor (a ≥ 0, b > 0)

4. Scoaterea factorilor de sub radical

√(a² · b) = a√b (a ≥ 0)
Pătratul perfect se scoate din radical

5. Introducerea factorilor sub radical

a√b = √(a² · b) (a ≥ 0)
Factorul se ridică la pătrat când intră sub radical

6. Raționalizarea numitorului

1/√a = √a/a
Se înmulțește numărătorul și numitorul cu √a

III. Exemple Rezolvate

Exemplul 1: Înmulțirea puterilor
Calculați: 2³ · 2⁴
Rezolvare:
Folosim formula: am · an = am+n
2³ · 2⁴ = 23+4 = 2⁷
2⁷ = 128
Răspuns: 128
Exemplul 2: Împărțirea puterilor
Calculați: 5⁸ / 5⁵
Rezolvare:
Folosim formula: am / an = am-n
5⁸ / 5⁵ = 58-5 = 5³
5³ = 125
Răspuns: 125
Exemplul 3: Putere la putere
Calculați: (3²)⁴
Rezolvare:
Folosim formula: (am)n = amn
(3²)⁴ = 32·4 = 3⁸
3⁸ = 6561
Răspuns: 6561
Exemplul 4: Simplificare radicali
Calculați: √50
Rezolvare:
Descompunem 50 = 25 · 2
√50 = √(25 · 2) = √25 · √2
√50 = 5√2
Răspuns: 5√2
Exemplul 5: Raționalizare
Raționalizați: 6/√3
Rezolvare:
Înmulțim numărătorul și numitorul cu √3:
6/√3 = (6 · √3)/(√3 · √3)
= 6√3 / 3
= 2√3
Răspuns: 2√3
