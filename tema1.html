Formule de Calcul Prescurtat

Algebră - Clasa a 7-a și a 8-a

I. Formulele de Bază

1. Pătratul sumei

(a + b)2 = a2 + 2ab + b2

2. Pătratul diferenței

(a - b)2 = a2 - 2ab + b2

3. Diferența pătratelor

a2 - b2 = (a - b)(a + b)

4. Cubul sumei

(a + b)3 = a3 + 3a2b + 3ab2 + b3

5. Cubul diferenței

(a - b)3 = a3 - 3a2b + 3ab2 - b3

6. Suma cuburilor

a3 + b3 = (a + b)(a2 - ab + b2)

7. Diferența cuburilor

a3 - b3 = (a - b)(a2 + ab + b2)

II. Exemple Rezolvate

Exemplul 1: Calculați (x + 3)2
Rezolvare:
Folosim formula: (a + b)2 = a2 + 2ab + b2
Unde a = x și b = 3
(x + 3)2 = x2 + 2·x·3 + 32
(x + 3)2 = x2 + 6x + 9
Răspuns: x2 + 6x + 9
Exemplul 2: Calculați (2x - 5)2
Rezolvare:
Folosim formula: (a - b)2 = a2 - 2ab + b2
Unde a = 2x și b = 5
(2x - 5)2 = (2x)2 - 2·2x·5 + 52
(2x - 5)2 = 4x2 - 20x + 25
Răspuns: 4x2 - 20x + 25
Exemplul 3: Descompuneți în factori x2 - 16
Rezolvare:
Observăm că avem diferența a două pătrate: x2 - 42
Folosim formula: a2 - b2 = (a - b)(a + b)
x2 - 16 = (x - 4)(x + 4)
Răspuns: (x - 4)(x + 4)
Exemplul 4: Calculați 492 - 1
Rezolvare:
Folosim formula diferenței pătratelor:
492 - 12 = (49 - 1)(49 + 1)
= 48 × 50
= 2400
Răspuns: 2400
Exemplul 5: Calculați (x + 2)3
Rezolvare:
Folosim formula: (a + b)3 = a3 + 3a2b + 3ab2 + b3
(x + 2)3 = x3 + 3·x2·2 + 3·x·22 + 23
= x3 + 6x2 + 12x + 8
Răspuns: x3 + 6x2 + 12x + 8
